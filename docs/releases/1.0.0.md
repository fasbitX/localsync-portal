# Release 1.0.0

**Date:** 2026-04-13
**Type:** Major - Initial Release

## Summary

Initial release of LocalSync Portal -- a photo publishing system for sports photographers.

## Features

- **Local client** (Node.js/Express) watches a photos directory and syncs files to a remote server
- **Remote server** (Java/Spring Boot) stores photos, serves public galleries via UUID links
- **Folder management**: Create, rename, delete folders with nested hierarchy support
- **Photo upload**: Drag-and-drop or right-click upload with progress bar
- **Interactive folder tree**: Expand/collapse, context menus, drag-and-drop onto folders
- **Sync engine**: Queue-based file sync with retry and exponential backoff (1s/3s/9s)
- **Contact & group management**: CRUD for contacts, groups, and group membership
- **Gallery invites**: Send email invitations with gallery links via SMTP
- **EXIF extraction**: Camera, lens, exposure, ISO, focal length metadata from photos
- **Admin dashboard** on remote server for managing folders and photos
- **API key authentication** for sync operations, BCrypt + session for admin

## Infrastructure

- PostgreSQL databases on both local and remote
- Chokidar file watcher for automatic sync triggering
- Multer for multipart file handling
- Nodemailer for email delivery
